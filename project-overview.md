Project Overview: StudyMaster
Problema
Estudiantes estudian ineficientemente: lectura pasiva sin retención, cero estructura, sin visibilidad de progreso, baja consistencia. Resultado: muchas horas, poco aprendizaje real.
Solución
App web que convierte contenido en planes de estudio personalizados con flashcards inteligentes (IA), repetición espaciada, Pomodoro integrado, y gamificación visual (heatmaps + streaks).
Features Core

Upload de material → generación automática de flashcards (conceptuales, contextuales, cronológicas)
Algoritmo de repetición espaciada
Timer Pomodoro con sesiones de estudio
Heatmap de consistencia estilo GitHub
Dashboard de progreso y analytics

Stack Tecnológico

Frontend: React Native (es para una app celular)
Backend: Python (Flask/FastAPI) + PostgreSQL
IA: OpenAI API (generación de flashcards)
Storage: AWS S3 (PDFs/documentos)
Auth: JWT/OAuth

Modelo de Negocio
Freemium:

Free: 50 flashcards, 1 materia, features básicos
Premium ($9.99/mes): Ilimitado + IA avanzada + analytics

Target
Estudiantes universitarios USA (18-24), expansión LATAM con pricing localizado.
